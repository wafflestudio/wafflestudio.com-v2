# wafflestudio.com-v2

### 설치

- `npm run prepare-husky`
  - `husky` 초기 세팅
  - `.husky/` 스크립트들에 실행 권한 부여

### 개발 컨벤션

| 속성   | 포맷          | 예시                                         |
| ------ | ------------- | -------------------------------------------- |
| branch | `feature/#\d` | `feature/#3`                                 |
| commit | `[#\d] *`     | `[#3] husky를 이용한 git commit prefix 세팅` |
